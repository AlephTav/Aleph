Aleph\\Net\\Request
===================



Общая информация
----------------

**Наследование:** нет

**Дочерние классы:** нет

**Интерфейсы:** нет

**Источник:** `lib/net/request.php`



Класс обеспечивает доступ к основным параметрам и заголовкам текущего
HTTP-запроса к серверу.



Общедоступные статические методы
--------------------------------

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**Параметры:** нет

Возвращает экземпляр класса.
